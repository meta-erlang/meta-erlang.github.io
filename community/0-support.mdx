# Support

The primary channel for any sort of issues is [the meta-erlang issue track](https://github.com/meta-erlang/meta-erlang/issues).

## Discussion forums {#discussion-forums}

The official channel is [the meta-erlang discussion forum](https://github.com/orgs/meta-erlang/discussions) at github project.

## Feature requests {#feature-requests}

For new feature requests, you can create a github issue at [github](https://github.com/meta-erlang/meta-erlang/issues).

## News {#news}

For the latest news about Docusaurus, [follow announcements on github](https://github.com/orgs/meta-erlang/discussions/categories/announcements).

Also check out the [meta-erlang blog](/blog) to stay in sync with the latest news.
